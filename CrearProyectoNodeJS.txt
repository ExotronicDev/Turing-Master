Dentro de la carpeta a trabajar poner : npm init -y
Despúes instalar todo lo demas: npm i mongoose express nodemon morgan concurrently axios dotenv colors jsonwebtoken bcryptjs cookie-parser slugify helmet xss-clean express-rate-limit hpp cors (Instalaciones básicas)

Para instalar React
Dentro de la misma carpeta se pone el comando : npx create-react-app my-app (donde my-app es el nombre de la carpeta a crear)
Dentro de la carpeta creada por React (cd my-app), poner los comandos : npm i sweetalert react-router-dom bootstrap js-cookie jwt_decoder (Instalaciones basicas) cdbreact (Para SideBar)

* En el caso de no instalar bootstrap con npm:
Para tener Bootstrap 5.1.3 dentro de react (Skins), ingresar a la carpeta public, luego en el archivo index.html, buscar la línea
dentro del <head> de "<title>React App</title>" (Por la linea 27), debajo de eso agregar este codigo:
